Tech Stack

This project uses the following technologies and frameworks:

  • Backend: Java 8, Spring Boot 2.7.18, Spring Security, Spring Data JPA, Hibernate
  • Frontend: PrimeFaces 11 (JSF), Thymeleaf, Thymeleaf Layout Dialect
  • Database: PostgreSQL (production), H2 (development/testing)
  • Build: Maven, JoinFaces 4.1.11
  • APIs: RESTful services, OAuth 2.0 integrations, 10+ market data source wrappers
  • AI/LLM: NVIDIA NIM integration for AI-powered insights
  • Caching: In-memory cache service with configurable TTL
  • Testing: JUnit 5, Mockito, Playwright (E2E)

Features

  • Recommendation for Ad Platform + Commerce Sector + Client combinations
  • See KPI metrics for recommendations
  • See current portfolios
  • See AI assisted recommendations for gaps in portfolio coverages
  • See breakdowns for each ad platform including KPI metrics
  • Return on Investment calculator for Ad Platform + Commerce Sector + Business Type

AI Models / Services

  • Coding Tool: Opencode
  • Service: Opencode Go, NVIDIA NIM
  • Models: deepseek-v4-flash, glm-5.2, deepseek-v4-pro, kimi-k2.6, qwen3.7-plus, GPT-OSS-120B, MiniMax-M2.7

Challenges

I faced a lot of issues with getting the dashboard to load as it was returning 500 errors upon initial build. Eventually the LLM agent figured out the issue and I was able to load it and start working through implementing and improving the requirements.

Initially, all of the data was mock data, so I found the correct API services to load some live data. Unfortunately a lot of the data sources are either paid-tier or very complex to implement in a short time.

I had to change the strategy of producing relevant KPI metrics to match to the data sources, so that was quite a bit of rework.

Overall, since I don't have domain knowledge for media buying, I had to both learn and imagine what the relevant information and relevant format of the information is for an analyst to give them insights that may be time consuming or difficult to obtain without the use of a structured aggregation of varied data inputs and AI analysis of those inputs.
